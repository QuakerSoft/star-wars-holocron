# 🌌 Star Wars Encyclopedia CLI

A Java-based command-line application that allows users to explore the Star Wars universe using real-time data from the SWAPI. The project also integrates AI-powered summarization to generate descriptions of selected entities such as characters, planets, species, and starships.


---
## 🚀 Features

- Search Star Wars characters, planets, species, and starships
- Fetch real-time data from the SWAPI (Star Wars API)
- AI-generated summaries for selected results
- Simple and interactive CLI menu interface

 ## 🛠️ Technologies Used

- Java
- SWAPI (Star Wars REST API)
- Google Gemini API (AI summarization)
- Java HttpClient
- JSON parsing
